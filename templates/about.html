About the Sugarcane Transcriptome Database

This local web resource collects transcript sequences from multiple Saccharum cultivars and related species (e.g. BC, Co11015, R570, SP). It is designed for fast, interactive exploration of large FASTA datasets directly from the browser.

The backend uses PostgreSQL to store headers and sequences, together with automatically computed features such as sequence length and GC content. The web interface is built with Flask and Bootstrap 5, providing a publication-ready layout similar to established plant databases.

Typical use-cases include:

  • Quick lookup of specific transcript IDs or headers.
  • Filtering transcripts by cultivar/species.
  • Subsetting by GC content or sequence length for downstream analyses.
  • Downloading sequences in FASTA format for further alignment or functional annotation.
